# Even Number Checker (Python)

This program takes an integer input and checks if the number is even.

## How it works
The program uses the modulus operator `%` to check if the number is divisible by 2.

If the remainder is 0, the number is even.

## Example

Input:
2

Output:
2 is even

Input:
3

Output:
3 is not even

- Python
- User input
- Conditional statements
